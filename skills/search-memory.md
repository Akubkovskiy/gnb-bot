# Навык: Поиск по базе знаний

## Триггер
Вопросы: "кто?", "когда?", "сколько?", "какая длина?"

## Источники
- gnb-transitions.json — переходы ГНБ
- people.json — специалисты
- projects.json — проекты
- organizations.json — организации

## Примеры
- "Кто проверяющий на ГНБ 15?" → gnb-transitions.json
- "Какая длина у ГНБ 11-11?" → L_план, L_профиль, L_общая
- "Кто технадзор в 3-м районе?" → people.json фильтр по роли+району
- "Сколько ГНБ по ЗИЛу?" → count transitions
