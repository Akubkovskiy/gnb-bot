# Навык: Создание нового комплекта актов ГНБ

## Триггер
Пользователь говорит: "Новый ГНБ", "Новый заказ", /new

## Алгоритм
1. Спросить заказчика (предложить из базы projects.json)
2. Спросить объект (предложить из папок заказчика)
3. Спросить номер ГНБ перехода
4. Проверить/создать директорию Работа\[Заказчик]\[Объект]\
5. Запросить документы (схема, акт осмотра, паспорт трубы, образец)
6. По мере получения — извлечь данные (OCR/чтение)
7. Показать все извлечённые данные, проверить подписантов
8. После подтверждения — заполнить шаблоны Excel
9. Сохранить, отправить в Telegram, обновить базу
