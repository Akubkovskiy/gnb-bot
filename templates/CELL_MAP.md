# Карта ячеек Лист1 — Акты ГНБ шаблон v2.xlsx (v3 layout)

Бот заполняет ТОЛЬКО Лист1. Остальные 10 листов обновляются автоматически через формулы `=Лист1!XX`.

## Правило пустых ячеек

Если данных нет (например, нет подписанта 3), бот записывает `" "` (пробел).
Формулы на листах подтянут пустую строку — визуально строка будет пустой при печати.

## Структура Лист1

### Секция ИДЕНТИФИКАЦИЯ (строки 2–11)

A = метка, B = значение.

| Ячейка | JSON-ключ | Назначение | Обяз. |
|--------|-----------|-----------|-------|
| B3 | title_line | Наименование объекта (КЛ ГНБ) | да |
| B4 | object_name | Название стройки / объект | да |
| B5 | address | Адрес | да |
| B6 | gnb_number | Номер ГНБ перехода | да |
| B7 | project_number | Номер/шифр проекта | да |
| B8 | start_date | Дата начала работ | да |
| B9 | end_date | Дата окончания работ | да |
| B10 | executor | Исполнитель (организация) | да |
| B11 | completion_date | Дата завершения работ | да |

### Секция ОРГАНИЗАЦИИ (строки 13–16)

| Ячейка | JSON-ключ | Назначение | Обяз. |
|--------|-----------|-----------|-------|
| B14 | customer | Заказчик | да |
| B15 | contractor | Подрядчик | да |
| B16 | designer | Проектировщик | да |

### Секция ПОДПИСАНТЫ (строки 18–23)

Таблица: A = роль, B = описание (орг + должн. + ФИО), C = подпись для актов (с чертой).

| Строка | Роль | B (описание) | C (подпись) | Обяз. |
|--------|------|-------------|-------------|-------|
| 20 | Подписант 1 (заказчик) | B20 | C20 | да |
| 21 | Подписант 2 (подрядчик) | B21 | C21 | да |
| 22 | Подписант 3 (опцион.) | B22 | C22 | нет |
| 23 | Тех.надзор (= подп.1?) | B23 | C23 | да |

Подписант 3 — опциональный (контролирующий орган, напр. АНО "ОЭК Стройтрест").
Если пусто (`" "`) — на листах будет пустая строка.

Бот спрашивает: «Помимо заказчика и подрядчика, есть ли третий подписант?»

### Секция ТРУБА (строки 25–27)

| Ячейка | JSON-ключ | Назначение | Обяз. |
|--------|-----------|-----------|-------|
| B26 | pipe_mark | Марка трубы (полная) | да |
| B27 | pipe_diameter | Диаметр трубы | да |

### Секция ПАРАМЕТРЫ ГНБ (строки 29–31)

Заголовки в строке 30, значения в строке 31.

| Ячейка | JSON-ключ | Назначение | Формула | Обяз. |
|--------|-----------|-----------|---------|-------|
| A31 | gnb_number_table | Номер ГНБ | — | да |
| B31 | plan_length | L план | — | нет |
| C31 | profile_length | L профиль | — | да |
| D31 | pipe_count | Кол-во труб | — | да |
| E31 | — | L трубы | =C31*D31 | авто |
| F31 | drill_diameter | d скважины | — | нет |
| G31 | configuration | Конфигурация | — | да |
| H31 | — | Стыки | =E31/13 | авто |
