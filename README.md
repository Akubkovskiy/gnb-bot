# Construction Docs Automation Bot

Telegram bot for automating construction documentation (Acts of Hidden Works, Statements) using Google Cloud Services.

## Features
- 📁 **Auto-sorting**: Upload files to Google Drive automatically.
- 👁️ **OCR**: Extract data from photos and PDFs.
- 🤖 **Assistant**: Interactive data validation via Telegram.
- 📄 **Generation**: Create Google Docs from templates and export to PDF/ZIP.

## Setup
1. Clone repo.
2. Setup `.env` with Telegram Token and Google Credentials.
3. Run `python bot.py`.
